# Prework Study Guide Webpage

## Description

- Q: What was my motivation to build a prework study guide?
- A: My motivation was to have a place to store for the information i deemed critical to my success.

- Q: Why did you build this project? (Note: the answer is not "Because it was a homework assignment.")
- A: I built this project to dip my toes in the world of web development and get a basic understanding of git.

- Q: What problem does it solve?
- A: I am now able to access my notes from anywhere.

- Q: What did you learn?
- A: My biggest takeaway from this prework is the importance of github and workflow.
